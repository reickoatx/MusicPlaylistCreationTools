How to build the Macintosh applications

Since most Macintosh users may not be familiar with the operation of running shell scripts in a terminal, the Playlist Creation Tools applications for the Macintosh were built using a special tool, provided by a third party.

Platypus is a developer tool for creating application wrappers around scripts. In other words, it creates Mac OS X applications that execute a script which is bundled within the application. Command line scripts can thus be run transparently from the graphical window environment, integrating seamlessly with the user interface

Platypus is written by Sveinbjorn Thordarson and you can visit him on the Platypus Product Home Page
